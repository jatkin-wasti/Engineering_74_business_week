# Bash and Linux commands

- Where am I
```
pwd
```

- How to create a Directory
```
mkdir name_of_dir
```

- How to create a file
```
touch name_of_file
```

- How to change Directory
```
cd dir_name
```

-How to come out of a Directory
```
cd ..
```

- Who am I
```
uname
```

- list Directories and all Directories
```
ls
```
